All Plugins

A list of plugins I have developed over the years, organised chronologically, with the most recent ones at the top.

2024

  • tmux-compile.nvim: nvim plugin to configure a dynamic run shortcut within TMUX.

  • only-tmux.nvim: nvim plugin to work with tmux to extend the functionality of :only.

  • tokyoburn.nvim: Neovim theme. Fork of folke/tokyonight in red/orange/yellow accent.

  • numericals.nvim: [WIP] nvim plugin to add basic math operation capabilities to the editor.
